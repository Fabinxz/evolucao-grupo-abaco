# SSC0104 - Evolução Histórica da Computação e Aplicações (2026): O Ábaco e o Soroban

Repositório do projeto focado na história e no uso do ábaco, desenvolvido para o Museu de Computação Prof. Odelar Leite Linhares no ICMC - USP São Carlos.

O projeto inclui uma linha do tempo interativa e um simulador prático do ábaco japonês (Soroban).
